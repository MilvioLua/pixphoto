Installation

    IMPORTANT: You must update from version 3.6, if you upgrade from another version will throw an error.
  • Upload the file v3.7.zip found inside the Update-v3.7 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v3.7.zip and enter at URL https://yoursite.com/v3.7/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v3.7/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.
  • Add this new text string in your file resources / lang / en / misc.php
    • // Version 3.7
'daily_limit_downloads' => 'Daily limit of downloads (free photos)',
'reached_daily_download' => 'You have reached the daily download limit',
'exclusive_author' => 'Exclusive Author',
'non_exclusive_author' => 'Non-exclusive author',
'exclusivity_items' => 'Exclusivity of your items',
'who_can_sell' => 'Who can sell',
'only_admin' => 'Only Admin',
'all_members' => 'All members',
'who_can_upload' => 'Who can upload',
'error_upload' => "You don't have permission to upload",
'show_counter' => 'Show counters on the homepage',
'show_categories_index' => 'Show categories on the homepage',
'allow_free_photos' => 'Allow free photos',
    If you have multiple languages on your site you should add to each file.
  • Copy the following files and folder to your current site:

    • vendor (Full folder)
    • composer.json
    • composer.lock
